# BEYOND THE MUSEUM

Cinematic Egypt tourism landing page built with **HTML**, **CSS**, **Bootstrap 5**, and minimal **JavaScript**.

![BEYOND THE MUSEUM](images/BEYOND%20THE%20MUSEUM.png)

## Overview

This project presents a storytelling journey:

1. Hero (start from the museum)
2. The Journey (historical timeline)
3. Museum Treasures
4. Explore Cairo (interactive image + list)
5. Best Time to Visit Egypt (seasonal pills + panel)
6. The Legacy

## Tech Stack

- HTML5
- CSS3
- Bootstrap 5
- Vanilla JavaScript

## Features

- Cinematic dark luxury design with warm gold accents
- Responsive layout (desktop/tablet/mobile)
- Interactive Explore Cairo section (hover/click updates image and content)
- EN/AR language switch with RTL support
- Smooth transitions and hover effects

## Run Locally

Open `index.html` directly in your browser, or use a local server (recommended).

Example with VS Code Live Server:

1. Open project folder
2. Right-click `index.html`
3. Click "Open with Live Server"
